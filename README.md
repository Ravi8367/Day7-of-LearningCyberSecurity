# Hacking Lab

A hacking lab is a safe, isolated digital environment (often virtual) where individuals or teams can practice ethical hacking, cybersecurity defense, and penetration testing by simulating real-world attacks and vulnerabilities without risking actual networks.

# Damn Vulnerablen Web Application

DVWA is a PHP/MySQL web application intentionally built with numerous security flaws, designed as a safe, legal platform for security professionals, developers, and students to practice, learn, and test web application security skills, including identifying vulnerabilities like SQL Injection, Cross-Site Scripting (XSS), and Command Injection.

# DVWA Installation

Install XAMPP Server - Download and install.

Launch XAMPP Control Panel

Start Apache and MySQL

Download DVWA and Extract to xampp/htdocs.

Configure DVWA: Rename
config.inc.php.dist to config.inc.php

Access DVWA: Type localhost in browser
Login: userename - admin, password - password
